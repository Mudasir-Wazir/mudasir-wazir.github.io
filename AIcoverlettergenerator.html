Project Overview

I developed an AI-powered cover letter generator using Cursor AI, DeepSeek AI, and Beautiful Soup, showcasing the power of LLMs (Large Language Models) combined with text analytics. This tool crafts personalized, professional cover letters tailored to specific job roles by simply inputting a job URL and resume.

Transitioning to Cursor AI from VScode proved to be a seamless process, enhancing my productivity with its intuitive interface and smart agentic coding features. Utilizing the Agent functionality in Cursor AI was particularly exciting during the development of my cover letter generator.

The primary aim of this project wasn't solely to create a cover letter tool but to demonstrate how AI tools like Cursor AI can streamline the development of such applications, making them more accessible to developers. This shift allows developers to focus on creativity and complex problem-solving while delegating repetitive tasks to AI technology.

The project was build on top of Meri Nova's work.
